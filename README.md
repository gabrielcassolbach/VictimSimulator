#-------------------------------------------------------------------#
to run the project, follow the steps bellow: 

1. create a virtual environment in python:
    - python3 -m venv .venv

2. enter in venv:
    - source .venv/bin/activate
    
3. install library dependencies:
    - pip install -r req.txt

4. if you install a library:
    - pip freeze > req.txt
